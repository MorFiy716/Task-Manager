# Task Manager

Веб-приложение для управления задачами.

## Функционал
- Создание, редактирование, удаление задач
- Фильтрация по статусу, приоритету, категории
- Статистика выполнения
- Поиск задач

# Установите зависимости:
- pip install -r requirements.txt
