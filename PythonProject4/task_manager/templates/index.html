Как получить пароль приложения

Для Gmail
  1. Включите двухфакторную аутентификацию
  2. Перейдите на страницу паролей приложений
  3. Создайте новый пароль для "Почта"
  4. Используйте его в поле "Пароль" выше
Для Яндекс
  1. Включите двухфакторную аутентификацию
  2. Создайте пароль приложения в настройках безопасности
  3. Используйте его в поле "Пароль"
